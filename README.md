# Med_Inventory 🏥

A web-based Medical Inventory Management System designed to help pharmacies or small clinics track their medicine stock, users, and dashboard analytics.

## 🚀 Features
* **Dashboard:** Overview of total medicines, categories, and stock status.
* **Medicine Management:** Add, Update, View, and Delete (CRUD) medicine records.
* **User Management:** Secure login and user creation system.
* **Clean UI:** Responsive design using modern CSS.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Advanced Java (Servlets/JSP or Spring Boot)
* **IDE:** Spring Tool Suite (STS 4)
* **Database:** MySQL & MongoDB

  
## 📂 Project Structure
* `/html`: Contains the structure of all pages (Dashboard, Login, etc.).
* `/css`: Custom styling for each module.
* `/js`: Functional logic for form handling and data display.
